### 使用說明
1.搜尋＋一個空格＋想搜尋的關鍵字 例如：搜尋 lemon <br>
2.機器人會告訴你搜尋的歌曲名稱和ID <br>
3.輸入 歌詞＋一個空格＋歌曲ID 例如：歌詞 11243 <br>
4.即可查詢到日文歌曲的中日歌詞 <br>
